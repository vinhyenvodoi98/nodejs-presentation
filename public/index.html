What is a computer network ?

A computer network is a set of computers connected together for the purpose of sharing resources

How many types of network are there ?

Geographical area

Nano : Nanoscale Networks

BAN : Body Area Networks

PAN : Personal Area Networks

LAN : Local Area Networks

CAN : Campus Area Networks

MAN : Metropolitan Area Networks

RAN : Radio Access Network

WAN : Wide Area Networks

Network protocol

• This is the language, or set of rules, that computer use to communicate

What is wireless networking ?

Use electromagnetic waves to transmit data

Main types of Wireless network

• Satellites - for long distances

• WiMax - for connecting Wi-Fi hotspots

• Wi-Fi - for medium-range distances

• Bluetooth - for short distances

• GSM - for mobile phones

Which is better, a wired or wireless LAN ?

• Wired LANs : Difficult to install ,but cheaper,faster and more reliable

• Wireless networks : let you move, or roam , but they are less secure and subject to interference
